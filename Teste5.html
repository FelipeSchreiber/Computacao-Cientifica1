Parte 1: Método do retângulo¶

Para calcular uma aproximao para uma integral, o mtodo do retngulo divide o intervalo de integrao em $N$ intervalos $I_i = [x_i,x_{i+1}]$, com $i=0,\ldots,N-1$. O erro parcial no intervalo $I_i$ definido como a diferena entre a aproximao do mtodo do retngulo e a integral exata neste intervalo.

  • uma lista (ou np.array) com a aproximao da integral em cada intervalo.

def rect_parcs(f, a, b, N):
    h = np.abs(a-b)/N
    init_points = np.linspace(a,b,N,endpoint=False)
    approx = []
    for init in init_points:
        approx.append(h*f(init))
    return init_points, approx

Student's answer Score: 2.0 / 2.0 (Top)

O grfico acima similar ao grfico da funo coseno. De fato, se analisarmos analiticamente, temos que o erro $e_k$ de cada intervalo dado por: $ \begin{equation} e_k = hf(x_k) - \int_{x_k}^{x_{k+1}} f(x) dx\\ e_k = \int_{x_k}^{x_{k+1}} f(x_k) - f(x) dx \end{equation} $ Fazendo a substituio de "x" por $x_k + ht$, temos que $dx = h.dt$ nessa nova substituio. Portanto:

$ \begin{equation} e_k = \int_{0}^{1} [f(x_k) - f(x_k + th)]h dt \end{equation} $

Sabendo ainda que a expanso de Taylor nos d: $ \begin{equation} f(x+\epsilon) = f(x) + f'(x).\epsilon + r_k(\epsilon)\\ f(x) - f(x+\epsilon) = - f'(x).\epsilon + r_k(\epsilon)\\ \end{equation} $

Substituindo a expanso a cima na integral: $ \begin{equation} e_k = \int_{0}^{1} [- f'(x)th + r_k(th)]h dt\\ \text{(h e f'(x) so constantes, logo podem sair da integral)}\\ e_k = -f'(x)h*h\int_{0}^{1}t dt + h\int_{0}^{1}r_k(th) dt\\ e_k = -\frac{f'(x)h^2}{2} + o(h^2)\\ \end{equation} $

Assim, vemos pela expresso do erro que a parte dominante proporcional a menos a derivada da funo que queremos calcular a integral multiplicado pelo h - tamanho do intervalo entre x's consecutivos - elevado ao quadrado. De fato, se plotarmos o grfico do erro esperado vemos que ele praticamente igual ao obtido empiracamente. Essa diferena se d pelo termo $o(h^2)$ que no foi levado em conta no clculo do erro predito.

O comportamento dessas curvas pode ser explicado fazendo-se a anlise do somatrio do erro:

$ \begin{equation} e_k = -\frac{f'(x)h^2}{2} + o(h^2)\\ e_total = \sum_{k = 0}^{k = N-1} -\frac{f'(x)h^2}{2} + o(h^2)\\ \end{equation} $

Contudo, se uma funo g(x) qualquer integrvel Riemann, ento temos que:

$ \begin{equation} \lim_{h \to 0}\sum_{k = 0}^{k = N-1}h g(x_k) = \int_a^b g(x) dx + o(1)\\ \end{equation} $

Como a funo f'(x) atende a essa condio, podemos substituir na equao do erro $ \begin{equation} e_{total} = \sum_{k = 0}^{k = N-1} -\frac{f'(x)h^2}{2} + o(h^2)\\ e_{total} = \int_{a}^{b} -\frac{f'(x)h^2}{2} dx + o(h) + o(1) \\ e_{total} = -\frac{(f(b) - f(a))h}{2} + o(h)\\ \end{equation} $

def trap_parcs(f, a, b, N):
    h = np.abs(a-b)/N
    init_points = np.arange(a,b,h)
    approx = []
    for init in init_points:
        approx.append(0.5*h*( f(init) + f(init + h) ) )    
    return init_points, approx
